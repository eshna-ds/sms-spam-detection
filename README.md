🚀 Overview - 
This project is a machine learning-based system that classifies SMS messages as Spam or Not Spam (Ham) 
using Natural Language Processing (NLP) techniques.
It helps in filtering unwanted messages and improving communication efficiency.

🎯 Problem Statement - 
Spam messages are a common issue in digital communication.
The goal of this project is to build a model that can automatically detect and classify spam messages based on their content.

📂 Dataset - 
The dataset contains labeled SMS messages as spam or ham
Commonly used dataset from Kaggle (SMS Spam Collection Dataset)

🛠️ Technologies Used - 
Python
Pandas & NumPy
Scikit-learn
Natural Language Processing (NLP)
Matplotlib / Seaborn (for visualization)

⚙️ Approach - 
1. Data Preprocessing
Removed unnecessary characters and symbols
Converted text to lowercase
Tokenization and cleaning

2. Feature Extraction
Used TF-IDF Vectorization to convert text into numerical features

3. Model Building
Applied machine learning algorithms such as:
Naive Bayes
Logistic Regression

4. Model Evaluation
Evaluated using:
Accuracy
Precision
Confusion Matrix

📊 Results
Achieved an accuracy 
Model performs well in identifying spam messages with high precision
